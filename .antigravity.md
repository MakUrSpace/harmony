# Antigravity Instructions

- DO NOT run `python` or `pytest` directly in the shell.
- ALWAYS use `nix develop -c python <script>` or `nix develop -c pytest` for running python code and tests.
- This ensures all necessary dependencies (like OpenCV / cv2) are properly loaded from the nix environment.
- Prefer `nix develop -c <command>` over bare commands for any development tasks in this repository.
